# Data Pipeline

```text
Public Credit Dataset
        │
        ▼
Cleaning
        │
        ▼
Validation
        │
        ▼
Feature Engineering
        │
        ▼
Synthetic Dataset
        │
        ▼

        │
        ▼

        │
```

The processed dataset is delivered to the AI Risk Agent, which calculates a Trust Score and assigns a Green, Yellow, or Red risk level for each member.
